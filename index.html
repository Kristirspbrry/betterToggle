betterToggle

Sleek CSS3 toggling of Elements.

Download

Single Element

code

 $("#click").click(function  () {
    $("example1").betterToggle();
});

Multiple Elements

Code

$("#click2").click(function  () {
    $(".example2").betterToggle();
})

Inserting Elements

$("#click3").click(function  () {
var insert = '<img style="display:none" src="images/holi3.jpg"';
$("#example3").append($(insert).betterToggle());
});

The inserting html should have display:none
